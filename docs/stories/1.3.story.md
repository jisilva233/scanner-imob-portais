# Story 1.3 — Dashboard Streamlit

## Story

**Como** usuário do scanner imobiliário,
**quero** abrir um painel web com `streamlit run dashboard/app.py` e filtrar anúncios por preço, bairro e metragem,
**para que** eu possa analisar o mercado imobiliário sem precisar escrever queries SQL.

## Status

Draft

## Acceptance Criteria

- [ ] **AC1:** `streamlit run dashboard/app.py` abre painel em `http://localhost:8501`
- [ ] **AC2:** Painel exibe métricas: total de anúncios, preço médio, número de portais, número de bairros
- [ ] **AC3:** Filtro de preço: slider de intervalo `(min, max)` formatado como `R$ X.XXX`
- [ ] **AC4:** Filtro de bairro: `multiselect` com todos os bairros únicos disponíveis
- [ ] **AC5:** Filtro de metragem: slider de intervalo `(min, max)` formatado como `XX m²`
- [ ] **AC6:** Filtro de portal: `multiselect` com ZAP / VIVAREAL / OLX
- [ ] **AC7:** Tabela exibe anúncios filtrados com coluna "Link" clicável
- [ ] **AC8:** Dados carregados via SQLAlchemy com `@st.cache_data(ttl=300)` (cache 5 min)
- [ ] **AC9:** Mensagem de aviso quando não há dados (`st.warning`)
- [ ] **AC10:** Layout usa `st.set_page_config(layout="wide")`

```python
import streamlit as st
import pandas as pd
from sqlalchemy import select
from sqlalchemy.orm import Session
from db.models import PropertyListingDB, engine

st.set_page_config(page_title="Imob Scanner", page_icon="🏠", layout="wide")
```

### Carregamento de Dados com Cache
[Source: squads/imob-scraper/agents/dashboard-agent.md]
```python
@st.cache_data(ttl=300)
def load_data() -> pd.DataFrame:
    with Session(engine) as session:
        rows = session.execute(
            select(PropertyListingDB).where(PropertyListingDB.is_active == True)
        ).scalars().all()
    return pd.DataFrame([...])
```

### Filtros — Sidebar
[Source: squads/imob-scraper/tasks/run-dashboard.md]

| Filtro | Widget | Config |
|--------|--------|--------|
| Preço (R$) | `st.sidebar.slider` | `format="R$ %,.0f"` |
| Bairro | `st.sidebar.multiselect` | `unique()` dos dados |
| Metragem (m²) | `st.sidebar.slider` | `format="%.0f m²"` |
| Portal | `st.sidebar.multiselect` | ZAP/VIVAREAL/OLX |

### Tabela com Link Clicável
[Source: squads/imob-scraper/tasks/run-dashboard.md]
```python
st.dataframe(
    filtered,
    column_config={
        "Link": st.column_config.LinkColumn("Link", display_text="Ver anúncio"),
        "Preço (R$)": st.column_config.NumberColumn(format="R$ %,.0f"),
    }
)
```

### Métricas de Resumo
```python
col1, col2, col3, col4 = st.columns(4)
col1.metric("Total de Anúncios", len(df))
col2.metric("Preço Médio", f"R$ {df['Preço (R$)'].mean():,.0f}")
col3.metric("Portais", df["Portal"].nunique())
col4.metric("Bairros", df["Bairro"].nunique())
```
