# Story 1.2 — Scrapers Playwright + Script Principal

## Story

**Como** usuário do scanner imobiliário,
**quero** executar `python scan_listings.py --city "São Paulo"` e ter anúncios dos três portais coletados e armazenados automaticamente no PostgreSQL,
**para que** eu tenha dados atualizados de imóveis sem acesso manual aos sites.

### Classe Base Abstrata
[Source: squads/imob-scraper/tasks/scrape-zap.md]

```python
# scrapers/base.py
from abc import ABC, abstractmethod
from db.models import PropertyListing

class BaseScraper(ABC):
    def __init__(self, rate_limit: float = 1.5):
        self.rate_limit = rate_limit

    @abstractmethod
    async def scrape(self, city: str, max_pages: int = 10) -> list[PropertyListing]:
        ...

    def city_to_slug(self, city: str) -> str:
        """Converte 'São Paulo' → 'sao-paulo'"""
        import unicodedata
        nfkd = unicodedata.normalize('NFKD', city)
        ascii_str = nfkd.encode('ASCII', 'ignore').decode()
        return ascii_str.lower().replace(' ', '-')
```
